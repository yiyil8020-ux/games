# 游戏合集

包含两个独立的 HTML 游戏，纯前端实现，无需服务器。

## 游戏列表

### 1. 小区风云 (Community Mystery)

社交推理游戏，你是一名新任业委会成员，需要在 7 天内调查物业经理失踪案。

**游戏特色**：
- 🔍 调查推理：收集线索，询问证人
- 🗣️ 对话系统：与 5 个 NPC 互动，建立好感度
- 📍 地点探索：搜索 10 个地点寻找证据
- 🧩 线索关联：组合线索发现真相
- 🔐 多结局：7 种不同结局

**NPC 角色**：
- 刘阿姨 - 广场舞队长，小区万事通
- 张工程师 - IT 工程师，监控专家
- 李姐 - 便利店老板娘，情报中心
- 赵老师 - 退休会计，账目专家
- 小陈 - 外卖骑手，神秘身份

**操作方式**：
- 点击 NPC 进行对话
- 点击地点进行搜索
- 使用推理板关联线索
- 最终指认凶手

**相关文件**：
- `community_mystery.html` - 游戏主文件
- `build_game.py` - 游戏生成器（完整版）
- `gen_game.py` - 游戏生成器（简化版）

---

### 2. 夏夜晚风 (Summer Night)

视觉小说游戏，讲述一个夏天、三个女孩的故事。

**游戏特色**：
- 🌸 日系视觉小说风格
- 🎵 动态音乐和音效
- 💕 好感度系统
- 🌻 花园养成系统
- 🌙 多结局（19种结局）

**角色介绍**：
- **苏念** - 住在对门的钢琴女孩，温柔安静
- **林小鹿** - 天台浇花的元气少女，活泼开朗
- **沈以诺** - 咖啡店老板，酷酷的短发女生

**游戏系统**：
- 好感度系统：与三位女主角互动提升好感
- 成长系统：通过选择提升个人成长值
- 花园系统：在天台种植花草
- 存档系统：支持 3 个存档槽位

**操作方式**：
- 点击屏幕推进剧情
- 选择对话选项影响好感度
- 空格/回车快速推进
- 支持自动播放和跳过已读

**相关文件**：
- `summer_night.html` - 游戏主文件

## 快速开始

直接在浏览器中打开游戏文件即可：

```bash
# 打开小区风云
open community_mystery.html

# 打开夏夜晚风
open summer_night.html
```

或者使用本地服务器：

```bash
python -m http.server 8000
# 然后访问 http://localhost:8000/community_mystery.html
# 或 http://localhost:8000/summer_night.html
```

## 技术特点

- 纯 HTML + CSS + JavaScript 实现
- 无需任何框架依赖
- 响应式设计，支持移动端
- 本地存储存档
- 流畅的动画效果
